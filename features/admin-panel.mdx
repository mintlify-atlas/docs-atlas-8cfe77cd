## Overview

The Admin Panel provides a secure web interface for managing users, issuing Bits recharges, and viewing platform analytics through leaderboards.

## Access & Authentication

### Route Configuration

The admin panel is accessible at `/admin` (`app.py:313-340`):

```python
@app.route('/admin', methods=["GET", "POST"])
def admin():
    mensaje = ""

    if request.method == "POST":
        telegram_id = request.form["telegram_id"]
        cantidad = int(request.form["cantidad"])
        password = request.form.get("password")

        ADMIN_PASSWORD = "admin"

        if password == ADMIN_PASSWORD:
            database.recargar_bits(telegram_id, cantidad)
            mensaje = f"✅ Éxito: Se recargaron {cantidad} Bits al usuario con ID {telegram_id}"
        else:
            mensaje = "❌ Error: Contraseña de administrador incorrecta."
```

<Warning>
**Security Notice:** The default password is `"admin"`. In production:
- Use environment variables: `os.getenv('ADMIN_PASSWORD')`
- Implement proper authentication (OAuth, JWT)
- Add rate limiting to prevent brute force
- Use HTTPS for all admin traffic
</Warning>

### Client-Side Validation

(`templates/admin.html:252-293`):

```javascript
window.adminConfirmSubmit = function (event) {
    var form = event.target;
    var telegram = form.telegram_id;
    var amount = form.cantidad;

    function hideError(id) { 
        document.getElementById(id).classList.add('visually-hidden'); 
    }
    function showError(id, msg) { 
        var el = document.getElementById(id); 
        el.textContent = msg; 
        el.classList.remove('visually-hidden'); 
    }

    hideError('telegram-error'); 
    hideError('cantidad-error');

    // Validate Telegram ID format
    var idValid = /^\d{5,20}$/.test(telegram.value.trim());
    if (!idValid) {
        showError('telegram-error', 'Telegram ID inválido. Introduce solo dígitos (5–20).');
        telegram.focus();
        event.preventDefault();
        return false;
    }

    // Validate amount
    var qty = Number(amount.value);
    if (!Number.isInteger(qty) || qty < 1) {
        showError('cantidad-error', 'Introduce una cantidad válida (entero mayor que 0).');
        amount.focus();
        event.preventDefault();
        return false;
    }

    // Confirmation dialog
    var confirmMsg = 'Vas a recargar ' + qty + ' Bits al usuario ' + telegram.value + '. ¿Confirmar?';
    if (!confirm(confirmMsg)) {
        event.preventDefault();
        return false;
    }

    return true;
};
```

```python
def obtener_top_recargas() -> list:
    with get_connection() as conn:
        rows = conn.execute("""
            SELECT telegram_id, nombre, username, photo_url, total_recargas
            FROM usuarios
            WHERE total_recargas > 0
            ORDER BY total_recargas DESC
            LIMIT 10
        """).fetchall()
        return [dict(row) for row in rows]
```
