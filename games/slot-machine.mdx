## Game Overview

The **Zona Jackpot 777 Slot Machine** is a luxury 5-reel slot game featuring premium PNG graphics, advanced physics animations, and a progressive betting system. Win up to **100x your bet** on diamond combinations!

<Note>
The slot machine uses a weighted symbol system with a 30% base win chance across all 5 reels, balanced for fair gameplay.
</Note>

## Symbol Types & Payouts

Each symbol has a unique weight (rarity) and payout multiplier:

| Symbol | Rarity | Multiplier | Description |

### Payout Calculation

```javascript
totalWin = bet × symbolMultiplier × lengthMultiplier
```

**Examples:**
- **3 Bells** (bet 10): `10 × 2 × 1 = 20 Bits`
- **4 Crowns** (bet 10): `10 × 15 × 3 = 450 Bits`
- **5 Diamonds** (bet 10): `10 × 100 × 10 = 10,000 Bits` 💎🎉

Winning triggers a **physics-based particle system** where coins:

2. Arc upward and spread laterally
3. Bounce realistically on the tray floor
4. Settle with friction and rotation

## Technical Details

### Reel Animation System

Each reel uses the **Web Animations API** for smooth, hardware-accelerated spinning:

```javascript
// From slotmachine.js:155-165
const anim = track.animate([
  { transform: 'translateY(0px)', filter: 'blur(0)', offset: 0 },
  { transform: `translateY(${targetY * 0.15}px)`, filter: 'blur(15px)', offset: 0.2 },
  { transform: `translateY(${targetY * 0.85}px)`, filter: 'blur(15px)', offset: 0.8 },
  { transform: `translateY(${targetY + (symbolHeight * 0.4)}px)`, filter: 'blur(0)', offset: 0.96 },
  { transform: `translateY(${targetY}px)`, filter: 'blur(0)', offset: 1.0 }
], {
  duration: duration,
  easing: 'cubic-bezier(0.1, 0.7, 0.1, 1)',
  fill: 'forwards'
});
```

**Key features:**
- Dynamic duration: Later reels spin longer (150ms delay per reel)
- Motion blur peaks at 20% and 80% of the animation
- Overshoot effect at 96% for realistic bounce
