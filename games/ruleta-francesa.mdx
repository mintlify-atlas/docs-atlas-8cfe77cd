## Game Overview

**Ruleta Francesa** (French Roulette) is the premium European roulette variant featuring:

- **Single zero (0)** wheel for better odds
- **37 numbers** (0-36)
- **Real-time physics** simulation
- **Multiple bet types** with payouts up to 35:1

<Note>
The single-zero wheel gives players a **2.7% house edge**, significantly better than American roulette's 5.26% double-zero variant.
</Note>

## The Roulette Wheel

### Number Layout

The French wheel arranges 37 numbers in a specific non-sequential pattern:

```
0 (green)
26-3-35-12-28-7-29-18-22-9-31-14-20-1-33-16-24-5-10-23-8-30-11-36-13-27-6-34-17-25-2-21-4-19-15-32
```

**Color distribution:**
- **Red**: 1, 3, 5, 7, 9, 12, 14, 16, 18, 19, 21, 23, 25, 27, 30, 32, 34, 36
- **Black**: 2, 4, 6, 8, 10, 11, 13, 15, 17, 20, 22, 24, 26, 28, 29, 31, 33, 35
- **Green**: 0

### Wheel Physics

The game uses advanced physics to simulate realistic wheel and ball behavior:

```javascript
// From physics.js - wheel rotation
wheelSpeed *= Math.pow(CONFIG.FRICCION_RULETA, dt * 60);
if (Math.abs(wheelSpeed) < CONFIG.VEL_MIN) wheelSpeed = 0;
State.wheelAngle += wheelSpeed * dt;
```

**Physics constants:**
- **Wheel friction**: `0.985` (slow deceleration)
- **Ball friction**: `0.92` (faster deceleration)
- **Minimum velocity**: `0.001` rad/s
- **Launch delay**: 800ms after wheel starts

  </Step>
  
  <Step title="Watch the Physics">

## Physics Deep Dive

### Ball Movement System

The ball follows a complex multi-phase trajectory:

<Steps>
  <Step title="Launch Phase">
    Ball starts at dispenser angle with high angular velocity:
    
    ```javascript
    // From main.js:167-168
    State.ballAngularSpeed = sign * Math.abs(wheelSpeed) * 
      CONFIG.IMPULSO_BOLA_FACTOR + 
      randRange(-CONFIG.IMPULSO_BOLA_RANDOM, CONFIG.IMPULSO_BOLA_RANDOM);
    ```
    
    - Opposite direction to wheel spin
    - ~1.8-2.2x wheel speed
    - Random variance for unpredictability
  </Step>
  
  <Step title="Spiral Inward">
    Radial position decreases gradually:
    
    ```javascript
    // Ball radius decreases as it loses energy
    State.ballRadius -= CONFIG.BALL_RADIAL_DECAY * dt;
    State.ballRadius = clamp(State.ballRadius, rMin, rMax);
    ```
    
    Creates the iconic roulette spiral effect.
  </Step>
  
  <Step title="Collision Detection">
    When ball radius reaches pocket zone:
    
    ```javascript
    const pocketRadius = rMin + (rMax - rMin) * 0.18;
    if (State.ballRadius <= pocketRadius + 0.5 && 
        Math.abs(State.ballAngularSpeed) < CONFIG.VEL_MIN) {
      // Ball has settled!
      const sectorIdx = sectorBajoBola(slice);
      const ganador = State.numeros[sectorIdx];
    }
    ```
  </Step>
  
  <Step title="Final Settlement">
    Both wheel and ball must stop:
    
    ```javascript
    if (Math.abs(wheelSpeed) < CONFIG.VEL_MIN &&
        Math.abs(State.ballAngularSpeed) < CONFIG.VEL_MIN &&
        State.ballRadius <= pocketRadius + 0.5) {
      finalizar();  // Determine winner
    }
    ```
  </Step>
</Steps>

### Sector Detection

```javascript
// From physics.js
export function sectorBajoBola(sliceAngle) {
  // Normalize ball angle to [0, 2π]
  let normalizedAngle = (State.ballAngle - State.wheelAngle) % (2 * Math.PI);
  if (normalizedAngle < 0) normalizedAngle += 2 * Math.PI;
  
  // Find which sector the angle falls into
  const sectorIndex = Math.floor(normalizedAngle / sliceAngle) % State.numeros.length;
  return sectorIndex;
}
```

## Tips & Strategies

<AccordionGroup>
  <Accordion title="The House Always Wins (Eventually)">
    With a 2.7% house edge, the casino has a mathematical advantage. Over many spins:
    
    ```
    Expected return = 97.3% of total bets
    Expected loss = 2.7% of total bets
    ```
    
    Short-term variance can create wins, but long sessions favor the house.
  </Accordion>
  
  <Accordion title="Even-Money Bets for Consistency">
    Red/Black, Even/Odd, High/Low all offer:
    - **48.65% win rate**
    - Smallest variance
    - Best for bankroll preservation
    
    Downside: Lowest excitement and payout potential.
  </Accordion>
  
  <Accordion title="Straight Bets for Jackpots">
    Pleno bets on single numbers:
    - **35:1 payout** is the highest
    - Only 2.7% hit rate (1 in 37)
    - Requires patience and large bankroll
    
    Consider betting **multiple numbers** (e.g., 5 different plenos) to increase hit rate while maintaining high payouts.
  </Accordion>
  
  <Accordion title="Avoid Betting Systems">
    Common systems like Martingale, Fibonacci, D'Alembert:
    - **Don't change house edge** (still 2.7%)
    - Require infinite bankroll to guarantee profit
    - Can lead to catastrophic losses during streaks
    
    Play for fun, not as a reliable income source.
  </Accordion>
  
  <Accordion title="Track Hot/Cold Numbers">
    The "Historial" shows recent results. Some players bet on:
    - **Hot numbers**: Recently appeared ("they're on a roll")
    - **Cold numbers**: Overdue ("law of averages")
    
    **Reality**: Each spin is independent. Past results don't affect future outcomes (gambler's fallacy).
  </Accordion>
  
  <Accordion title="Set Win/Loss Limits">
    Before playing, decide:
    - **Win goal**: Cash out after gaining X Bits
    - **Loss limit**: Stop after losing Y Bits
    
    Stick to these limits to avoid chasing losses or giving back winnings.
  </Accordion>
</AccordionGroup>

## Game Configuration

From `physics.js`:

```javascript
export const CONFIG = Object.freeze({
  IMPULSO_RULETA: 8.5,           // Initial wheel spin speed
  FRICCION_RULETA: 0.985,        // Wheel deceleration (98.5% per frame)
  IMPULSO_BOLA_FACTOR: 1.8,      // Ball speed relative to wheel
  IMPULSO_BOLA_RANDOM: 0.4,      // Random variance in ball launch
  FRICCION_BOLA: 0.92,           // Ball deceleration (faster than wheel)
  BALL_RADIAL_DECAY: 0.08,       // How fast ball spirals inward
  BALL_R_MAX_RATIO: 0.85,        // Starting radius (85% of wheel)
  BALL_R_MIN_RATIO: 0.35,        // Pocket radius (35% of wheel)
  VEL_MIN: 0.001,                // Threshold for "stopped"
  LAUNCH_DELAY: 800              // ms before ball launches
});
```
